OC_ttt
======

<b>Requirements</b>

- Xcode 5.0 or later
- xctool

<b>Tic Tac Toe Engine</b>

# To run tests:

- `cd ttt_engine`
- `xctool -workspace OCTicTacToe.xcworkspace -scheme Specs`

<b>Tic Tac Toe in Console</b>

# To run tests:

- `cd console_ttt`
- `xctool -workspace console_ttt.xcworkspace -scheme specs`

<b>To play from terminal</b>

- `./console_ttt/playGame`
